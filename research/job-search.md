# Job Search Notes — C++ in Games (US)

Criteria locked in: **US-based**, **full-time OR contract**, **any C++ in games**
(best-fit lean: engine / rendering / physics / systems).

> ⚠️ Tooling caveat: the web session's fetcher is blocked (HTTP 403) by job
> aggregators and studio ATS pages (Workday/Greenhouse/Lever/Workable) and by
> LinkedIn. So specific posting links rot fast and can't be auto-validated from
> the web session. Validate in a logged-in browser. Use the portals + alerts below.

## Salary benchmarks (US, 2026)
- Remote game developer avg ~$108k ($81k–$124k typical).
- Gameplay programmer ~$129k–$143k.
- General C++ (non-games, runs higher): mid $120k–$155k, senior $155k–$210k.
  → Games pays a discount vs general C++ — weigh that.
